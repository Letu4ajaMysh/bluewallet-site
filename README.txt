BlueWallet Ultra — структура проекта
====================================

Файлы:
- index.html                 — главная страница
- css/styles.css             — стили
- js/main.js                 — взаимодействия (particles, tilt, modal)
- assets/hero.jpg            — заглушка изображения кошелька (замени своей)
- assets/hero-video.mp4      — (опционально) фоновое видео - положи свое в assets
- assets/click.mp3           — (опционально) звук клика

Инструкция:
1) Распакуйте архив и откройте index.html локально или разверните на хостинге (Netlify, Vercel, GitHub Pages).
2) Замените assets/hero.jpg на свою качественную фотографию кошелька (1200–2400px).
3) При желании замените assets/hero-video.mp4 — проверьте расширения webm/mp4.
4) Настройте контакты: в index.html замените tel: и ссылки на WhatsApp/Telegram на реальные.
5) Для приёма заказов подключите backend: в js/main.js обработчик sendOrder — отправляйте POST на ваш endpoint или интегрируйте с сервисом (Zapier, Make, Google Forms, Airtable).

Поддержка:
- Хочешь 3D-модель (glTF) — я могу подготовить пример с three.js.
- Нужен ZIP с мини-ресурсами (иконки, логотип) — скажи, соберу.

Спасибо! Скажи, когда будешь готов скачать — я подготовлю архив и дам ссылку.
